## Опыт проектов

### Aeranta (2025 – настоящее время)
[https://aeranta.net](https://aeranta.net)  

Личный веб-проект метеостанции с прогнозом погоды, астрономическими данными, характеристиками луны, алертами и уведомлениями.

**Достижения:**
- Разработал структуру Django-приложения с интеграцией нескольких API (Telegram bot, OpenWeather, Auroras.live, IPGAstronomy).  
- Создал удобный пользовательский интерфейс, понятный даже не техническим пользователям.
- Настроил почтовый сервер.
- Настроил систему уведомлений с помощью телеграм-бота и почтового сервера. 
- Настроил автоматизацию задач с Celery для выполнения периодических задач(проверка состояния погоды, отработка алертов, отправка уведомлений).  
- Разработал гибкую архитектуру для лёгкого добавления новых API и алертов.  
- Создал адаптивный интерфейс карты с OpenLayers и тайлами CARTO Voyager.  
- Полностью контейнеризировал проект с помощью Docker для локальной и продакшен-развертки.  
- Получил практические навыки JavaScript и продолжаю развивать фронтенд-навыки.
